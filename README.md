# Zap Gadgets

A modern gadget store website showcasing the latest tech products with a clean design, responsive layout, and easy navigation for an enhanced shopping experience.

---

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/ThatoNdlovu049/Gadget-Store.git
2. Navigate into the project directory:
cd Gadget-Store
3. Open index.html in your browser to view the site.
## Usage
-Open the index.html file in any modern browser.
-Navigate through categories, featured products, and gadget listings.

Examples:

-Homepage: Displays featured gadgets and promotions.
-Product Page: Shows product details, images, and specifications.
-Navigation Bar: Quickly access different sections of the store.
## Features
-Responsive layout for desktop, tablet, and mobile devices
-Clean, modern design
-Easy navigation with intuitive menus
-Tech showcase highlighting the latest gadgets
-Interactive elements with smooth animations and transitions

## Configuration Options
-Theme Customization: Modify style.css to adjust colors, fonts, and layout.
-Product Data: Update product listings in index.html or connect to a backend API.
-Navigation Links: Configure menu items in the HTML header for custom categories.

## Troubleshooting
-Page not loading correctly: Ensure you are opening index.html in a modern browser (Chrome, Edge, Firefox).
-CSS not applied: Check that style.css is correctly linked in the HTML file.
-JavaScript not working: Verify that script.js is included before the closing </body> tag.
